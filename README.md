# Promoter-Predictor
An ML Model designed to predict the promoter strength (logarithmic scale from 1 to 5) of a 80pb promoter submited sequence.
Made using the EVMP architecture from *Yang et al. 2023* (https://www.biorxiv.org/content/biorxiv/early/2023/03/04/2022.10.15.512354.full.pdf) and trained the model using the database obtained from *Zhou et al. 2025* (https://academic.oup.com/nar/article/53/16/gkaf863/8246948)
